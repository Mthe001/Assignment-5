What is the Document Object Model (Dom) ?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a webpage in a way that programming languages, like JavaScript, can interact with. Essentially, it transforms the contents of a webpage (like HTML and XML documents) into a structured tree that programs can modify.

How do you select an element from DOM ?

To select an element from the DOM in JavaScript, you can use various methods provided by the document object. These methods allow you to select elements based on different criteria, such as their tag name, class name, ID, or more complex CSS selectors.

What is event delegation in the context of the DOM, and why is it useful ?

Event delegation is a technique in the DOM that allows you to handle events at a higher level in the DOM hierarchy, instead of attaching event listeners to specific individual elements. In essence, you attach a single event listener to a parent element, which listens for events triggered by its child elements. This works because of event bubbling, a process where events propagate up the DOM tree from the target element to its ancestors.
